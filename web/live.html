Detect → Restore Loop
🔍 Detect — LSTM scores risk every 10 requests on real telemetry
↪ Reroute — risk ≥ 0.65 or CB opens → all traffic to backup
🧠 Diagnose — rule engine classifies: cascade / overload / timeout / error_surge
🔧 Fix — throttle / circuit_break / backoff / isolate_dependency applied
✅ Restore — canary 10% → 50% → 100% once primary health confirmed
