06 · Self-Improving Pipeline

Detect drift. Augment. Retrain. Promote.

A closed loop that audits live telemetry, augments minority cases, retrains conditionally and only promotes the new checkpoint if it beats the incumbent. Runs nightly. Logs every decision.

01
◐
Audit
Roll a 7-day window. Compute KS-statistic per signal, per-API failure rate, sequence-length drift.
02
◑
Detect
Flag drift if KS > 0.1 on ≥ 3 signals, or AUC degradation ≥ 1.5% on holdout.
03
◒
Augment
SMOTE-ish injection for failure-class shortfalls. Synthetic precursor windows on demand.
04
◓
Retrain
Warm-start from last best. 10 epochs. Lower LR. Same focal-loss, same per-h pos_weight schedule.
05
◉
Promote
Promote only if Δ-AUC ≥ +0.3% AND conformal coverage holds. Otherwise revert + log.
